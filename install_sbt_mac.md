### Mac 平台安装 SBT

#### 通过第三方的包安装

    注意：第三方的包可能没有提供最新版本，可以将相关任何问题反馈给包相关的维护者

**通过 [Macports](http://macports.org/) 安装**

```
$ port install sbt
```

**通过 [Homebrew](http://mxcl.github.com/homebrew/) 安装**

```
$ brew install sbt
```

#### 通过通用的包安装

下载 [ZIP][down-zip] 包或 [TGZ][down-tgz] 包解压

#### 手动安装
